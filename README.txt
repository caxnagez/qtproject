# Книга рецептов с TTS, удалением и редактированием

Приложение для поиска, добавления, удаления и редактирования рецептов с возможностью озвучки инструкций при двойном клике. Используется pyttsx3 для озвучки.

## Техническое задание

- Поиск рецептов по списку ингредиентов
- Добавление новых рецептов с фото
- Удаление существующих рецептов
- Редактирование рецептов (название, ингредиенты, инструкция, фото) — при двойном клике
- Озвучка инструкции по приготовлению (TTS) при двойном клике на рецепт или кнопке "Озвучить"
- Озвучка не блокирует интерфейс, можно запускать несколько раз подряд
- Работа с локальной базой данных SQLite
- Просмотр списка ингредиентов при выборе рецепта
- Подсказка в поле поиска: "перечислите ингредиенты с маленькой буквы через запятую"

## Технологии

- PyQt6
- SQLite
- pyttsx3 (Text-to-Speech)

## Установка и запуск

1.  Установите Python 3.8+
2.  Клонируйте проект
3.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
4.  Убедитесь, что у вас есть:
    - `resources/def.png`
    - `ui/main.ui`
    - `ui/newrecipe.ui`
    - `ui/editrecipe.ui`
    - `data/initial_ingredients.txt`
    - `data/initial_recipes.txt`
5.  Запустите:
    ```bash
    python recipe.py
    ```

## Сборка standalone (.exe)

```bash
pip install pyinstaller
python -m PyInstaller --onefile --windowed --add-data "ui;ui" --add-data "resources;resources" --add-data "recipes.db;." --hidden-import=pyttsx3 --hidden-import=sip recipe.py
